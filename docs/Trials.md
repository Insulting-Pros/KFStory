This is a silly mod inspired by Trials in UT2004. It features objectives you have to go through, record speed runs, rankings based on speed runs, double / multi jumping support and dodging / walldodging support. By interface it is slightly like UT2004 Assault Â© [Marco](https://steamcommunity.com/profiles/76561197975509070/)

```clike
Game=TrialsMod.TrialGame
```
